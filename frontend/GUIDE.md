# CETS 前端應用 - 使用指南

## 快速演示步驟

### 1. 安裝和運行

```bash
# 進入前端目錄
cd frontend

# 安裝依賴（首次運行）
npm install

# 啟動開發服務器
npm run dev
```

應用會自動打開瀏覽器窗口，默認地址是 `http://localhost:5173`

### 2. 登入和初始體驗

系統預設已自動登入為 **「一般員工」**（EMPLOYEE 角色），可以：
- 查看所有活動列表
- 點擊活動卡片查看詳情
- 點擊「立即報名」按鈕報名活動
- 點擊右上角用戶菜單進入「個人資料」頁面

### 3. 各頁面功能說明

#### 首頁 - 活動列表 (`/`)
- **頁面功能**：
  - 顯示所有熱門活動
  - 支持按狀態篩選（全部、報名中、進行中、已結束）
  - 顯示活動的基本信息（時間、地點、報名人數、可用名額）
  - 點擊卡片進入詳情頁

- **互動操作**：
  - 選擇狀態篩選器重新加載活動列表
  - 點擊「查看詳情」進入活動詳情頁
  - 懸停卡片可看到過渡動畫

#### 活動詳情 (`/events/:eventId`)
- **頁面功能**：
  - 查看完整活動信息（標題、描述、時間、地點）
  - 顯示活動進度條（已報名人數 / 總容納人數）
  - 顯示票價資訊
  - 如果有空位，顯示「立即報名」按鈕

- **報名流程**：
  1. 點擊「立即報名」按鈕
  2. 填寫報名表單（姓名、電郵、電話、公司）
  3. 點擊「確認報名」
  4. 報名成功後自動跳轉到個人資料頁查看票券

#### 個人資料 (`/profile`)
- **頁面功能**：
  - 顯示當前登入用戶信息（姓名、郵件、部門、角色）
  - 分兩個 Tab：
    - **我的票券**：顯示已報名活動的票券卡片
    - **我的報名**：顯示報名記錄清單

- **票券功能**：
  - 點擊票券卡片查看完整詳情
  - 在詳情 Modal 中查看 QR Code
  - 支持複製票券號碼和下載票券（占位符）

### 4. 角色切換（開發模式）

在頁首有「角色切換」下拉菜單：
- **員工視圖** (EMPLOYEE) - 推薦用於演示
- **管理員視圖** (ADMIN) - 管理活動
- **驗票員視圖** (VERIFIER) - 掃描票券

切換角色會使用不同的 API Token 重新登入。

### 5. 測試數據

系統使用後端的演示數據。測試用戶信息：

**EMPLOYEE (一般員工)**
- 用戶 ID: `01E2EEMPLOYEETSMCROLEXXXXX`
- 員工 ID: `E2EEMP01`
- 姓名: `e2e-employee`
- 工作地點: `HSINCHU`

**活動數據**
- 後端已預先創建演示活動
- 可直接報名（如果有空位）
- 報名成功會生成票券

## 常見操作

### 查看所有活動
1. 訪問首頁 `/`
2. 選擇篩選條件（可選）
3. 活動卡片自動加載

### 報名活動
1. 進入活動詳情頁
2. 點擊「立即報名」按鈕
3. 填寫必要信息
4. 確認提交
5. 自動跳轉至個人資料頁查看票券

### 查看已報名活動
1. 點擊右上角用戶菜單
2. 選擇「個人資料」
3. 查看「我的報名」Tab

### 查看票券
1. 進入個人資料頁
2. 點擊「我的票券」Tab（默認選中）
3. 點擊票券卡片查看詳情和 QR Code

## 如果遇到問題

### 頁面無法加載
- 確認後端 API 服務正在運行 (`https://cets.alanh.uk`)
- 檢查瀏覽器控制台是否有錯誤信息
- 嘗試刷新頁面

### 無法報名
- 檢查活動是否已報滿
- 檢查活動狀態是否為「報名中」
- 確認表單所有必填欄位已填寫

### 票券無法顯示
- 確認已成功報名該活動
- 嘗試刷新「我的票券」頁面
- 檢查後端是否已生成票券
